# iwalle-cli
![Npm 版本](https://img.shields.io/badge/iwalle--cli-v0.0.8-blue)

## 📖简介

-  💡用于快速搭建前端项目的命令行工具，用于个人测试、学习

## 📑模版

-  ✅快速生成Vue3+TS+Vite+ElementPlus+Pinia+Axios+Mock 模版

## 🔧安装

```bash
# 使用模版创建新项目
iwalle create
 
# 根据项目文件夹名称创建新项目
iwalle create [projectName]

# 查看脚手架版本
iwalle -v
iwalle --version

# 更新脚手架
iwalle update
```
